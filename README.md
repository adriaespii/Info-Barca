# FCB Portal Blaugrana - 1º DAW ⚽🔵🔴

Benvinguts al meu **Projecte Intermodular (PI)** per al primer any del cicle Grau Superior en Desenvolupament d'Aplicacions Web (DAW).

Aquesta web és una aplicació frontal (*Single Page Application*) dedicada a informar sobre l'actualitat i estat de la plantilla oficial del FC Barcelona que respon exclusivament i en detall a les exigències acadèmiques.

## Estructura i Com executar el projecte
L'ambient d'execució i l'estructura interna està pensada per a no requerir de la utilització del Terminal o consoles. 

1. Baixat aquest repositori al teu equip de treball.
2. Com que implementem funcionalitats modernes asíncrones de JS al navegador per obtenir el JSON extern (`fetch`), es demana que per evadir els bloquejos restrictius del model CORS intern (*file://*) s'executi a través exclusivament del complement **Live Server (Visual Studio Code)** prement Click Dret sobre l'esborrany `index.html` > *Open with Live Server*.
3. Gaudeix de l'entorn.

## Característiques base del TFG:
- 🏥 **Control Mèdic**: Pestanya on veuràs automàticament tots els integrants donats de baixa en creuar-hi els paràmetres JSON.
- ⭐️ **Sistema de Favorit (Local Storage)**: Prem l'estrella del teu preferit a les diferents fixetes blanques dels perfils gràfics esportius; comprova com en refrescar un altre cop el projecte amb f5 (Reload) al Live Server la decisió es persisteix.
- 📉 **Mural d’Estadístiques (Ordenats amb MAP)**: S'ensenyarà una ordenació de Màxim Golejador en format Taula HTML d'acord amb valors matemàtics ordenats via script base.
